# vscode-pdf-plus

**Fork of vscode-pdfviewer** - Display PDF files in VSCode.

## Features

- Built-in PDF viewer
- Navigation tools  
- Custom scroll mode support
- **Delete PDF pages functionality**:

  - Delete multiple pages by selection
